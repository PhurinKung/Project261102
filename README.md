# Project C++ for 261102 - COMPUTER PROGRAMMING

Calendar GUI Application using c++ and Dear ImGui

### Features

* **Event Management :** add/edit/delete any events
* **Search Events :** search by title, detail, location, category (case-insensitive)
* **Upcoming Events :** show top 8 upcoming event (all or each category)
* **Auto-Save Data** 
* **Auto-Delete Old Events**
* รองรับภาษาไทย
